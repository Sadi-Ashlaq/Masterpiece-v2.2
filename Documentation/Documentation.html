This website is built with Laravel 7.7.1. I have also added few reusable components which speedup the development.

Project Information

This idea came up to me because I'm an expert in mobile software and hardware, most people have difficult time in getting the necessary help to fix their phones, my service will be their number one to go.

General Description

  • The website is made to help customers book appointment to fix their mobile, we will come to their doorstep after that
  • The user need to fill a form with the problem they are facing, they must be logged in
  • The user will be asked for a location so we can get to them
  • The user will get a bill after the phone is fixed
  • The website will have 3 types of users, Admin and super admin also normal user
  • Normal user will be able to register/login then use our service
  • Super admin will be able to control everything on the website and have access to every data
  • Admin will be able to control everything on the website and have access to every data

Main Goal

Lessen the time and effort for the client and come up with a trendy way to fix your phone.

Installation

Given below are the steps you need to follow, to install the Project on your system:

Step 1: Open the terminal in your root directory(project directory) & to install the composer packages run the following command:

composer install

Step 2: By running the following command, you will be able to get all the dependencies in your node_modules folder:

npm install

Step 3: To serve the application you need to run the following command in the project directory. (This will give you an address with port number 8000.) Now navigate to the given address you will see your application is running.

php artisan serve

To change the port address, run the following command:

php artisan serve --port=8080 // For port 8080
